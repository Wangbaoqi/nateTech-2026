## 第一章：底层逻辑与机制深化

### 1.1 真的在“赌”吗？(CTF 与 Order Book)
Polymarket 不是赌场，它是一个**去中心化交易所 (DEX)**。
- **技术底层**: 基于 Gnosis 的 **Conditional Tokens Framework (CTF)**。所有的预测份额本质上是 ERC-1155 代币。
- **流动性**: 采用 **CLOB (中央限价订单簿)**，这意味着你在和**人**（或做市商机器人）交易，而不是和智能合约（AMM）交易。
    - *启示*: 流动性不是无限的。大额买入会产生**滑点 (Slippage)**，必须学会看深度图 (Depth Chart)。

### 1.2 裁决的博弈 (The UMA Oracle)
UMA 预言机是“乐观机制”。
- **正常流程**: 提案 -> 2小时公示期 -> 无异议 -> 确认。
- **争议流程**: 如果有人对结果存疑（比如事件描述模糊），可以质押代币发起挑战。此时进入投票环节。
- **实战风险**:
    - *文字游戏*: 比如“比特币在周五前达到 $100k”，如果是周五 23:59:59 达到算不算？必须仔细阅读 **"Rules" (规则)** 选项卡。
    - *案例*: 曾有关于“NFT 地板价”的预测，因数据源（OpenSea vs Blur）不同导致巨大争议。**永远以规则中的“Resolution Source”为准，而不是你的常识。**

## 第二章：分板块实战策略 (Sector Strategies)

不同的板块，驱动价格的因素完全不同。

### 2.1 政治板块 (Politics) - "流动性之王"
这是 Polymarket 最大的板块（如美国大选）。
- **驱动力**: 民调 (Polls)、突发丑闻、辩论表现。
- **策略**: **“修正偏差” (Bias Correction)**
    - *现象*: 早期市场往往被极端情绪主导（如某些候选人的死忠粉无视数据买入），导致赔率与专业民调（如 538, Nate Silver）脱节。
    - *操作*: 当 Polymarket 赔率 (e.g., 60%) 显著高于聚合民调概率 (e.g., 45%) 时，**做空 (买入 No)**。
- **案例**: 2020/2024 大选中，特定州的“红市蜃楼”现象（计票顺序导致早期一方领先），熟悉计票规则的交易员在恐慌盘中捡到了便宜筹码。

### 2.2 加密板块 (Crypto) - "内幕与链上数据"
预测币价、空投、ETF 审批、黑客归还资金等。
- **驱动力**: 链上数据、Github 更新、SEC 公告。
- **策略**: **“关联性交易” (Correlation Play)**
    - *逻辑*: 某些事件与币价高度绑定，但反应有滞后。
    - *操作*: 比如“ETH ETF 是否通过”。如果彭博社分析师将概率上调至 90%，但 Polymarket 还在 60%，这就是机会。
    - *高级*: 利用币价波动对冲。比如你做多 "ETH > $3000"，同时在合约市场做空 ETH，赚取波动率溢价。

### 2.3 宏观经济 (Macro/Fed) - "与美联储博弈"
预测降息/加息、CPI 数据。
- **驱动力**: 经济数据、CME FedWatch 工具。
- **策略**: **“CME 套利”**
    - *原理*: 传统金融市场 (CME FedWatch) 的预测通常比 Polymarket 更“聪明”且流动性更好。
    - *操作*: 当 Polymarket 的“降息概率”与 CME 出现 >5% 的偏差时，无脑跟随 CME 的方向进行套利。Polymarket 的散户往往反应较慢。

### 2.4 流行文化/体育 (Pop Culture/Sports)
预测泰勒·斯威夫特订婚、超级碗冠军。
- **驱动力**: 社交媒体热度、八卦新闻。
- **策略**: **“情绪反转”**
    - *特点*: 这是一个纯散户市场，极易受谣言影响。
    - *操作*: 这里的 "No" 往往被低估。大众喜欢买 "Yes"（期待事情发生），导致 "Yes" 溢价。长期在这个板块**卖出热门事件的 Yes**（即买入 No）通常具有正期望值 (Positive EV)。

## 第三章：进阶操作技术

### 3.1 仓位管理：凯利公式 (Kelly Criterion)
不要梭哈。预测市场本质是概率游戏。
- **公式**: $f = (bp - q) / b$
    - $f$: 投注比例
    - $b$: 赔率 (1/胜率 - 1)
    - $p$: 你认为真实的胜率
    - $q$: 失败概率 (1-p)
- **实战**: 如果你认为某事发生概率是 80%，而市场定价 $0.60（赔率不错），不要全仓。计算凯利值（通常建议使用 **半凯利 Half-Kelly** 以降低波动风险）。

### 3.2 退出机制：不仅是持有到期
新手才持有到期，老手都在波段交易。
- **获利了结**: 你以 $0.20 买入，涨到 $0.80。此时风险收益比已经变了。如果事件还有变数，**卖出本金**或者全部止盈是明智的。
- **止损**: 当基本面发生根本变化（如你押注的候选人退选），立即止损，不要抱有幻想。

## 第四章：大量使用案例 (Case Studies)

### 案例 A：比特币 ETF 获批 (2024年初)
- **背景**: 市场在猜测 SEC 是否批准 BTC ETF。
- **信号**: 
    1.  SEC 官方推特被黑，发布假消息 -> 市场剧烈波动。
    2.  彭博社 ETF 分析师 Eric Balchunas 持续发布积极信号。
- **操作**: 
    - 早期：在 $0.20 左右埋伏 Yes。
    - 中期：当 Matrixport 发布看空报告导致币价大跌时，Polymarket 价格也随之下跌。这是**加仓良机**，因为基本面（SEC流程）并未改变。
    - 结果：ETF 通过，价格归 $1.00。

### 案例 B：Sam Altman 被解雇又回归 OpenAI
- **背景**: OpenAI 董事会突发解雇 Sam Altman。市场出现预测“Sam 是否会回归 CEO”。
- **信号**: 微软介入、员工联名信。
- **操作 (高频)**: 这是一个典型的新闻驱动市场。
    - 员工联名信发布 -> 这是一个极强的回归信号 -> 迅速买入 Yes。
    - 谈判破裂传闻 -> 迅速卖出或对冲。
- **教训**: 这种市场必须 24小时盯盘或设置提醒，局势瞬息万变。

### 案例 C：美联储议息 (CME 搬砖)
- **背景**: 预测“美联储 3月是否降息”。
- **观察**: CME 联邦基金利率期货显示降息概率仅为 10%。
- **机会**: Polymarket 上仍有一群死多头将价格维持在 25%。
- **操作**: 买入 "No" (成本 $0.75)，实际获胜概率 90%。这是一笔胜率极高、甚至可以说是“捡钱”的交易。
