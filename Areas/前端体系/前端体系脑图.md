
> 该结构旨在构建全景式的前端认知地图
## 1. 编程语言核心 (Languages Core)
- **JavaScript (ES Next)**
    - **底层原理**
        - 执行上下文 (Execution Context) & 调用栈 (Call Stack)
        - 作用域链 (Scope Chain) & 闭包 (Closure)
        - `this` 绑定机制 (Call/Apply/Bind)
        - 原型与原型链 (Prototype Chain)
        - 事件循环 (Event Loop): Macro/Micro tasks
        - 垃圾回收机制 (Garbage Collection): 标记清除, 引用计数
    - **异步编程**
        - Callback / Promise / Async & Await
        - Generator / Iterator
    - **函数式编程**
        - 纯函数 / 柯里化 / 组合 / 高阶函数
- **TypeScript**
    - **类型系统**
        - 基础类型 / 接口 (Interface) / 类型别名 (Type)
        - 泛型 (Generics)
        - 高级类型: `Pick`, `Omit`, `Record`, `Partial`, `Readonly`
        - 条件类型 (`extends ? :`) & `infer` 推断
        - 协变与逆变 (Covariance & Contravariance)
    - **工程实践**
        - `tsconfig.json` 详解
        - 声明文件 (`.d.ts`) 编写
        - 装饰器 (Decorators)
- **HTML5 & Web APIs**
    - 语义化标签
    - DOM / BOM 操作
    - Web Components (Shadow DOM)
    - PWA (Service Workers, Manifest)
    - Web Workers & Shared Workers
    - IndexedDB / LocalStorage / SessionStorage
- **CSS3 & 样式工程**
    - **布局系统**: Flexbox, Grid, Multi-column
    - **核心概念**: 盒模型, BFC/IFC, 层叠上下文 (Z-index), 优先级
    - **预/后处理器**: Sass/Less, PostCSS (Autoprefixer)
    - **架构方案**: CSS Modules, CSS-in-JS (Styled-components, Emotion), Atomic CSS (Tailwind)
    - **动画与特效**: Transition, Animation, Transform (2D/3D)

## 4. 浏览器与性能优化 (Browser & Performance)
- **浏览器渲染原理**
    - HTML Parsing -> DOM Tree
    - CSS Parsing -> CSSOM Tree
    - Render Tree -> Layout (Reflow) -> Paint (Repaint) -> Composite
- **性能指标 (Web Vitals)**
    - LCP (Largest Contentful Paint)
    - FID (First Input Delay) / INP (Interaction to Next Paint)
    - CLS (Cumulative Layout Shift)
    - FCP, TTI
- **优化手段**
    - **加载优化**: 代码分割 (Code Splitting), 懒加载 (Lazy Load), 预加载 (Preload/Prefetch)
    - **渲染优化**: 虚拟列表 (Virtual List), 防抖节流, requestAnimationFrame
    - **网络优化**: HTTP/2, CDN, Gzip/Brotli, 缓存策略 (Cache-Control)
    - **图片优化**: WebP, AVIF, 响应式图片
