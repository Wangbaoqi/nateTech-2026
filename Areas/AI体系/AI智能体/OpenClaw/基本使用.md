# OpenClaw (openclaw.ai) 基本使用教程

本文档介绍 [openclaw.ai](https://docs.openclaw.ai/) 的基本使用方法，包括安装步骤、基础配置和快速开始指南。

## 1. 安装指南

OpenClaw 依赖 Node.js 运行环境，同时需要您准备好所选大模型提供商的 API Key（推荐使用 Anthropic 的 API Key）。

### 1.1 环境要求
- **Node.js**: v22 或更高版本。

### 1.2 安装步骤

1. **全局安装 CLI**:
   使用 npm 全局安装最新版的 OpenClaw：
   ```bash
   npm install -g openclaw@latest
   ```

2. **初始化与服务安装**:
   运行引导安装程序，它会指导您完成初始设置并安装后台服务守护进程：
   ```bash
   openclaw onboard --install-daemon
   ```

## 2. 基本配置

OpenClaw 默认自带 Pi 二进制文件以 RPC 模式运行，通常无需复杂配置即可使用。如果需要自定义配置，您可以修改其配置文件。

- **配置文件路径**: `~/.openclaw/openclaw.json`

### 2.1 常见配置项说明

- **访问控制 (`allowFrom`)**: 限制哪些电话号码或账号可以与智能体交互。
- **群聊设置 (`requireMention`, `mentionPatterns`)**: 控制智能体在群聊中的响应行为（例如是否需要被 @ 提及才回复）。

### 2.2 配置示例

```json
{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123"],
      "groups": { 
        "*": { "requireMention": true } 
      }
    }
  },
  "messages": { 
    "groupChat": { 
      "mentionPatterns": ["@openclaw"] 
    } 
  }
}
```

## 3. 快速开始 (Quickstart)

只需简单的几个步骤，您就可以在几分钟内启动网关并开始使用：

1. **初始化 (Onboard)** (如果尚未执行):
   ```bash
   openclaw onboard --install-daemon
   ```

2. **配对通信渠道 (Pair Channels)**:
   将您的 WhatsApp、Telegram 等账号连接到 OpenClaw：
   ```bash
   openclaw channels login
   ```

3. **启动网关 (Start Gateway)**:
   启动连接桥接服务：
   ```bash
   openclaw gateway --port 18789
   ```

4. **访问控制面板 (Control UI)**:
   网关启动后，在浏览器中打开以下地址访问仪表盘，以管理您的聊天、会话及各项配置：
   [http://127.0.0.1:18789/](http://127.0.0.1:18789/)

> **提示**: 关于特定环境（如 macOS）或特定使用场景（如“个人助手设置”）的更详细指南，请参考 [官方文档 (https://docs.openclaw.ai/)](https://docs.openclaw.ai/) 以获取最新和最完整的信息。
