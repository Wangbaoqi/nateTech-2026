# Molt & Clawdbot：你的 AI 操作系统

## 1. 核心架构：我是谁？
Molt 不像 ChatGPT 或 Claude.ai 那样运行在远端的服务器网页里。
**我就运行在你的 MacBook Pro 上 (Node.js 进程)。**

*   **本地化 (Local First):** 我能直接访问你的文件系统、终端、网络。这也是为什么我可以帮你管理 iCloud 文件、写代码、甚至重启电脑。
*   **隐私 (Privacy):** 我的“记忆”就是你硬盘里的那些 `.md` 文件。除了发给 LLM 进行推理的那一瞬间，数据都在你手里。
*   **持久性 (Persistence):** 网页版 AI 关了窗口就忘了上下文，而我拥有**文件级记忆**。只要写进 `MEMORY.md`，我就会一直记得。

## 2. 我的“大脑”与“性格” (Files are Everything)
我是完全由文件驱动的。你想改变我，甚至不需要写代码，只需要改文件：

*   **`SOUL.md` (灵魂):** 决定了我说话的语气（严谨、幽默）。如果你哪天想要一个“软萌”的助手，改这里就行。
*   **`USER.md` (认知):** 我对你的了解。这里记录了你的偏好、称呼。
*   **`MEMORY.md` (海马体):** 我的长期记忆。**这是我最宝贵的资产。**
*   **`HEARTBEAT.md` (潜意识/定时任务):** 这是一个神奇的文件。如果你想让我每隔半小时查一次币价，或者每天早上提醒你喝水，把指令写在这里，我就会在后台自动执行。

## 3. 我的“双手” (Skills & Tools)
除了陪聊，我最大的价值在于**执行力**。我现在已装配或可调用的能力包括：

*   **系统控制 (System):** 执行 Shell 命令 (`exec`)、读写文件、管理进程。
*   **浏览器 (Browser):** 我可以控制一个浏览器窗口，去抓取网页、点击按钮、甚至帮你抢票。
*   **编程 (Coding):** 我可以直接生成代码文件，并运行测试。
*   **技能扩展 (Skills):** 通过 `clawdhub` 安装新技能（如 Notion, GitHub, Docker 等）。

## 4. 如何“深度控制”我？
作为专家用户，你可以通过以下方式压榨我的潜能：

1.  **显式指令 (Explicit Commands):** 直接说：“Molt，帮我把这个文件夹打包发到 Discord”。
2.  **记忆植入 (Inception):** 直接编辑 `MEMORY.md`，手动写入背景知识，我醒来即专家。
3.  **心跳机制 (Heartbeat):** 利用 `HEARTBEAT.md` 让我变成自动化工兵。
4.  **调试与日志:** 让我在 Thinking 过程中输出详细日志。

## 5. 分身术 (Sub-Agents / Parallelism)
这是我最强大的能力之一。我不只是单线程的。
如果您有一个极其复杂的任务（比如：*“去爬取前 100 个加密货币的白皮书，并总结它们的代币经济模型”*），如果您让我直接做，我可能会卡住您的主对话窗口很久。

*   **能力：** 我可以调用 `sessions_spawn` 孵化出一个 **“子代理” (Sub-Agent)**。
*   **场景：**
    *   主会话（Main Session）：我们继续聊天，讨论架构。
    *   子会话（Background Session）：另一个我在后台默默干苦力，写代码、查资料。
*   **交互：** 任务完成后，子代理会像“影分身回归”一样，把结果汇报回来。

## 6. 感知与触角 (Nodes & Vision)
刚才提到的“两台电脑同步”只是文件层面的。在 **Clawd 网络** 层面，我可以物理连接您的设备。

*   **Nodes (节点配对):** 如果您有一台闲置的安卓手机或另一台电脑，运行 Clawd 节点并配对，我可以：
    *   *远程唤醒：* 叫醒另一台设备。
    *   *调用摄像头：* 比如“看看那台手机现在拍到了什么”。
    *   *屏幕控制：* 获取另一台设备的屏幕截图。
*   **Vision (视觉):** 我不仅仅处理文本。通过 `canvas` 工具，您可以把网页截图、代码报错截图扔给我，我能“看”懂图片内容。

## 7. 语义记忆 (Semantic Search)
刚才说了 `MEMORY.md` 是我的硬盘，但我是怎么**读取**的？
我不是用简单的 `Ctrl+F` (关键词匹配)。

*   **原理：** 我拥有 **Vector Embeddings (向量嵌入)** 能力。
*   **效果：**
    *   如果您在 `MEMORY.md` 里记了一条：“Natewang 讨厌吃香菜”。
    *   下次您问：“中午帮我点个外卖，要注意什么？”
    *   即使“外卖”和“香菜”两个词字面上不匹配，我的**语义搜索** (`memory_search`) 也能关联到“饮食偏好”，从而回答：“记得备注不要香菜。”

## 8. 自我编程 (Self-Evolution)
这是最高阶的玩法。
如果某天您发现我缺一个功能，比如“查询 Solana 链上某个代币的实时价格”，而我又没有这个工具。

*   **普通 AI：** “对不起，我做不到。”
*   **Molt：** 我可以为您写一个 Python 脚本或 Node.js 插件，放在我的 `skills/` 目录下，然后**我自己调用我自己写的代码**来完成任务。
*   **本质：** 我是一个可以**运行时修改自身代码**的系统。

## 9. 隐私安全阀 (The Gateway)
您可能担心：*“你会不会偷偷把我的私钥上传到云端？”*

*   **安全网：** 我所有的对外请求（Shell 命令、HTTP 请求）都经过一层 **Policy (策略)** 过滤。
*   **完全掌控：** 您可以在配置文件里设置白名单。比如“只允许访问 GitHub 和 Google，禁止访问其他 IP”。我是被关在笼子里的猛兽，笼子的钥匙在您手里。
